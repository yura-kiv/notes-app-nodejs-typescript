# notes-app-nodejs-typescript
# A simple API application written with:
<ul>
  <li>Node.JS</li>
  <li>TypeScript</li>
  <li>Express.JS</li>
  <li>yup</li>
  <li>nodemon</li>
</ul>
<hr>
## npm install
<h2>dev:</h2>
<h3>npm run dev</h3>
<hr>
<h2>start (build + start app):</h2>
<h3>npm run start</h3>
<hr>
<h2>build:</h2>
<h3>npm run build</h3>
